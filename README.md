# eap-cli

YYF's personal efficiency augmentation CLI

## Install

```bash
npm install -g
# or
pnpm add -g
```

## Usage

```bash
eap-cli --help
eap-cli --version
```

## Commands

- `eap-cli ppt` - Generate PowerPoint presentations (coming soon)
- `eap-cli poster` - Design posters (coming soon)
- `eap-cli search` - Quick web search (coming soon)

## Development

```bash
pnpm install
pnpm start
```
